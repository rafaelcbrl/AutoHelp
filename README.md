# AutoHelp

## Sobre o projeto

O AutoHelp é uma plataforma de assistência veicular desenvolvida para a disciplina Projeto Integrador.

O sistema conecta clientes que precisam de ajuda com prestadores de serviços automotivos, como mecânicos, guinchos e entrega de combustível. Além disso, conta com uma Inteligência Artificial Automotiva para auxiliar na identificação preliminar de problemas em veículos.

---

## Funcionalidades

- Cadastro de usuários
- Login
- Perfil de usuário
- Solicitação de Mecânico
- Solicitação de Guincho
- Solicitação de Entrega de Combustível
- Chat entre cliente e prestador
- Compartilhamento de localização
- Histórico de atendimentos
- Avaliações por estrelas
- Inteligência Artificial Automotiva integrada com Gemini
